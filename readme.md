
## 📝 Personal Organization
Um sistema completo de organização pessoal para gerenciar tarefas e produtividade, construído com as tecnologias mais modernas do ecossistema React.

## 🚀 Tecnologias Utilizadas
Este projeto utiliza o seguinte conjunto de tecnologias:
React com Vite para um ambiente de desenvolvimento ultra-rápido.
TypeScript para garantir segurança e tipagem estática.
Tailwind CSS para estilização utilitária e design responsivo.
Clerk para autenticação robusta e gestão de usuários.
React-Toastify para notificações interativas e amigáveis.
Redux Toolkit (opcional/conforme solicitado) para gerenciamento de estado global.

## ✨ Funcionalidades
Autenticação Segura: Login, Cadastro e Perfil de usuário via Clerk.
Gestão de Tarefas: Adicionar, editar e excluir itens da sua rotina.
Feedback Visual: Notificações em tempo real para ações do usuário.
Design Responsivo: Adaptado para dispositivos móveis e desktop.
Modo Dark/Light: Estilização otimizada via Tailwind.

## 📦 Como Instalar e Rodar
Clone o repositório:
bash
git clone https://github.com
cd personal-organization
Use o código com cuidado.

Instale as dependências:
bash
npm install
# ou
yarn add
Use o código com cuidado.

Configure as Variáveis de Ambiente:
Crie um arquivo .env.local na raiz do projeto e adicione suas chaves do Clerk:
env
VITE_CLERK_PUBLISHABLE_KEY=pk_test_xxxxxxxxxxxxxxxx
Use o código com cuidado.

Inicie o servidor de desenvolvimento:
bash
npm run dev
Use o código com cuidado.
